# 操作系统
  ## 基本概念
  目标: 向上层提供方便易用的服务。<br />
  定义: 控制和管理计算机系统的软件和硬件资源，并合理地组织调度工作和资源来给用户提供接口和环境，是最基本的系统软件。
  功能: 文件管理，内存管理，CPU管理，设备管理
  特征: 并发，共享，虚拟，异步
  发展和阶段：
    1. 手工操作阶段
    2. 批处理阶段(单道批处理系统, 多道批处理系统)
    3. 分时操作系统
    4. 实时操作系统(硬实时系统, 软实时系统)
    5. 网络操作系统
    6. 分布式操作系统
    7. 个人计算机操作系统
